# 树与图的数学理论

## 一、图论基础

### 1.1 图的基本定义

**图**（Graph）是一个有序对 $G = (V, E)$，其中：
- $V$ 是**顶点集**（Vertex Set），元素称为**顶点**（vertex）或**节点**（node）
- $E$ 是**边集**（Edge Set），元素是 $V$ 中无序对（无向图）或有序对（有向图）

$$V = \{v_1, v_2, ..., v_n\}, \quad E \subseteq V \times V$$

**基本术语**：

| 术语 | 定义 | 记号 |
|------|------|------|
| 顶点数/阶 | 图中顶点的个数 | $n = \|V\|$ |
| 边数 | 图中边的条数 | $m = \|E\|$ |
| 邻接 | 两顶点之间有一条边相连 | $u \sim v$ |
| 关联 | 边与其端点之间的关系 | $e = uv$ |
| 环 | 连接同一顶点的边 | $e = vv$ |
| 重边 | 两个顶点之间多条边 | — |

### 1.2 图的分类

```
                        ┌─────────────┐
                        │     图      │
                        └──────┬──────┘
                               │
               ┌───────────────┼───────────────┐
               ▼               ▼               ▼
           无向图            有向图        混合图
               │               │
               ▼               ▼
           简单图          多重图
```

| 分类 | 定义 | 边方向 | 允许环 | 允许重边 |
|------|------|--------|--------|---------|
| **无向图** | 边无方向，$uv = vu$ | 无 | 可 | 可 |
| **有向图** | 边有方向，$uv \neq vu$ | 有 | 可 | 可 |
| **简单图** | 无环且无重边 | 皆可 | 否 | 否 |
| **多重图** | 允许重边 | 皆可 | 可 | 可 |
| **完全图** | 任意两顶点间恰有一条边 | 皆可 | 否 | 否 |
| **二分图** | 顶点可划分为两个集合，边只在集合之间 | 无向 | 否 | 否 |
| **平面图** | 可画在平面上边不相交 | 皆可 | — | — |
| **加权图** | 每条边附有权值 | 皆可 | — | — |

**经典例子：完全图 $K_5$**

$K_5$ 是 5 个顶点的完全无向图，边数为 $\binom{5}{2} = 10$，每个顶点的度均为 4。$K_5$ 是**非平面图**的最小例子之一（Kuratowski 定理）。

### 1.3 顶点度

**定义**：顶点 $v$ 的**度**（degree）$d(v)$ 是与其关联的边的数目。

$$d(v) = |\{u \in V : uv \in E\}|$$

**有向图的度**：
- **出度** $d^+(v)$：从 $v$ 出发的边数
- **入度** $d^-(v)$：到达 $v$ 的边数
- $d(v) = d^+(v) + d^-(v)$

**经典定理：握手引理（Handshaking Lemma）**

$$\sum_{v \in V} d(v) = 2|E|$$

即所有顶点的度数之和等于边数的两倍。对于有向图：
$$\sum_{v \in V} d^+(v) = \sum_{v \in V} d^-(v) = |E|$$

**推论**：任何图中度数为奇数的顶点个数为偶数。

## 二、树的理论

### 2.1 树的定义与基本性质

**定义**：**树**（Tree）是一个连通且无环的无向图。换言之，树是最小连通图——删除任意一条边就会使图不连通。

**树的等价定义**（以下条件互相等价）：

1. $G$ 是连通且无环的
2. $G$ 中任意两顶点之间有且仅有一条路径
3. $G$ 是连通的且 $|E| = |V| - 1$
4. $G$ 是无环的且 $|E| = |V| - 1$
5. $G$ 是连通的且删除任意一条边都会使图不连通（极小连通）
6. $G$ 是无环的且添加任意一条边都会产生环（极大无环）

**经典例子：6 个顶点的树**

```
    v₁ ── v₂ ── v₃
          │
          v₄ ── v₅ ── v₆
```

- $|V| = 6$, $|E| = 5 = 6 - 1$ ✓
- 任意两顶点间有唯一路径（如 $v_1$ 到 $v_6$ 的唯一路径为 $v_1-v_2-v_4-v_5-v_6$）✓
- 连接无直接边的两顶点（如 $v_3$ 和 $v_6$）会产生环 ✓

### 2.2 树的拓扑性质

| 性质 | 表述 |
|------|------|
| **叶节点** | 度数为 1 的顶点。任何非平凡树至少有两个叶节点 |
| **直径** | 树中最长路径的长度（边数） |
| **中心** | 所有顶点中离心率最小的顶点。一棵树的中心要么是一个顶点，要么是两个相邻顶点 |
| **半径** | 所有顶点的最小离心率，$r(T) = \min_{v} \max_{u} d(v,u)$ |
| **树心** | 通过反复删除叶节点最终得到的单个顶点或相邻两顶点 |

**经典例子：树的中心**

```
    A ── B ── C ── D ── E ── F
```

- 直径：$d(A,F) = 5$
- 删除所有叶节点 $(A,F)$ → $B \!-\! C \!-\! D \!-\! E$
- 再删除叶节点 $(B,E)$ → $C \!-\! D$
- 中心为 $\{C, D\}$，半径为 $\lceil 5/2 \rceil = 3$

### 2.3 特殊类型的树

#### 2.3.1 二叉树

**定义**：每个节点最多有两个子节点（左子节点和右子节点）的有根树。

**性质**：
- 深度为 $h$ 的二叉树最多有 $2^{h+1} - 1$ 个节点
- 第 $i$ 层最多有 $2^i$ 个节点（根为第 0 层）
- 有 $n$ 个节点的二叉树高度至少为 $\lfloor \log_2 n \rfloor$

**经典例子：完全二叉树**

```
          1
         / \
        2   3
       / \ / \
      4  5 6  7
```

- 所有层均满
- 节点数 $n = 2^{h+1} - 1 = 2^{3} - 1 = 7$
- 叶节点数 = $(n+1)/2 = 4$

#### 2.3.2 二叉搜索树（BST）

**定义**：对任意节点，其左子树所有节点的值小于该节点值，右子树所有节点的值大于该节点值。

**性质**：中序遍历 BST 得到递增序列。

```
          8
         / \
        3   10
       / \    \
      1   6    14
         / \   /
        4  7  13
```

#### 2.3.3 生成树

**定义**：图 $G$ 的**生成树**（Spanning Tree）是 $G$ 的一个子图，它包含 $G$ 的所有顶点且是一棵树。

**Cayley 公式**：$n$ 个顶点的完全图 $K_n$ 有 $n^{n-2}$ 棵不同的生成树。

**经典例子：$K_3$ 的生成树**

```
    1          1          1
   / \        /          \
  2   3      2 — 3       2 — 3
```

$K_3$ 有 $3^{3-2} = 3$ 棵不同的生成树。

#### 2.3.4 最小生成树（MST）

**定义**：在加权连通图中，所有生成树中边权之和最小的那棵称为**最小生成树**。

**构造算法**：

| 算法 | 策略 | 时间复杂度 | 核心思想 |
|------|------|-----------|---------|
| **Kruskal** | 贪心选最小边 | $O(m \log m)$ | 不断选取不形成环的最小权边 |
| **Prim** | 从顶点出发扩张 | $O((n+m)\log n)$ | 从一点出发，每次选最近的未访问顶点 |
| **Borůvka** | 并行选择 | $O(m \log n)$ | 每轮每个连通分量选最小出边 |

**经典例子：Kruskal 算法**

```
边集（按权重排序）：
A-B: 2,  A-C: 3,  B-C: 1,  B-D: 4,  C-D: 5

执行：
1. B-C (1)  ← 选 ✓
2. A-B (2)  ← 选 ✓
3. A-C (3)  ← A-B-C 已连通，跳过
4. B-D (4)  ← 选 ✓
5. C-D (5)  ← 已全连通，结束

MST 总权 = 1 + 2 + 4 = 7
```

### 2.4 树的编码与遍历

#### 2.4.1 Prüfer 序列

**定理**（Cayley）：$n$ 个顶点的有标号树与长度为 $n-2$ 的 Prüfer 序列一一对应。

**构造**（树 → Prüfer 序列）：
1. 找到编号最小的叶节点，将其邻接顶点加入序列
2. 删除该叶节点
3. 重复直到只剩两个顶点

**经典例子**：

```
树：
    2 ── 1 ── 3
         │
         4
         │
         5

执行：
删除最小叶 2，邻接顶点 1 → 序列: [1]
删除最小叶 3，邻接顶点 1 → 序列: [1, 1]
删除最小叶 4，邻接顶点 1 → 序列: [1, 1, 1]
删除最小叶 5，邻接顶点 4 → 序列: [1, 1, 1, 4]
Prüfer 序列 = [1, 1, 1, 4]
```

#### 2.4.2 树遍历

| 遍历方式 | 顺序 | 经典应用 |
|---------|------|---------|
| **先序** | 根 → 左 → 右 | 序列化树结构 |
| **中序** | 左 → 根 → 右 | BST 得有序序列 |
| **后序** | 左 → 右 → 根 | 表达式求值 |
| **层序** | 逐层从左到右 | 最短路径（未加权） |

## 三、图论核心理论

### 3.1 路径与连通性

#### 3.1.1 路径与回路

| 术语 | 定义 |
|------|------|
| **路径** | 顶点序列 $v_1v_2...v_k$ 满足相邻顶点间有边，且顶点不重复 |
| **迹** | 顶点可重复但边不重复 |
| **回路** | 起点和终点相同的路径 |
| **圈** | 至少 3 个顶点且无重复顶点的回路 |
| **欧拉回路** | 经过每条边恰好一次并回到起点的回路 |
| **哈密顿回路** | 经过每个顶点恰好一次并回到起点的回路 |

**经典例子：柯尼斯堡七桥问题（欧拉, 1736）**

```
         ╱ ╲        ╱ ╲
    陆地A     陆地B
         ╲ ╱        ╲ ╱
    ═══════════════════════
         ╱ ╲        ╱ ╲
    陆地C     陆地D
         ╲ ╱        ╲ ╱
```

欧拉证明：当且仅当每个顶点的度数均为偶数时，存在欧拉回路。七桥问题中四个顶点度数均为奇数，故不存在欧拉回路。这被认为是图论的起源。

#### 3.1.2 连通性

**定义**：
- **连通图**：任意两顶点间存在路径
- **连通分量**：极大连通子图
- **割点**：删除后连通分量数增加
- **桥**：删除后连通分量数增加的边
- **点连通度** $\kappa(G)$：使图不连通所需删除的最小顶点数
- **边连通度** $\lambda(G)$：使图不连通所需删除的最小边数

**Whitney 不等式**：对任何非平凡图 $G$：
$$\kappa(G) \leq \lambda(G) \leq \delta(G)$$

其中 $\delta(G)$ 为图的最小度。

**经典例子：3-连通图**

```
    A ── B ── C
    │    │    │
    D ── E ── F
    │    │    │
    G ── H ── I
```

删除任意 2 个顶点图仍连通（3×3 网格图是 3-连通的），但删除 3 个顶点就可能不连通。所以 $\kappa(G) = 3$。

#### 3.1.3 强连通与弱连通（有向图）

| 概念 | 定义 |
|------|------|
| **强连通** | 任意两顶点 $u,v$ 之间既有 $u \to v$ 路径又有 $v \to u$ 路径 |
| **弱连通** | 忽略方向后是无向连通图 |
| **强连通分量（SCC）** | 极大强连通子图 |

**经典例子：网页排名与强连通**

Web 的链接结构可以建模为有向图。Google 的 PageRank 算法利用强连通分量分析来评估网页重要性。Kosaraju 算法和 Tarjan 算法可在 $O(n+m)$ 时间内找出所有 SCC。

### 3.2 图的矩阵表示

#### 3.2.1 邻接矩阵

**定义**：$A(G)$ 是一个 $n \times n$ 矩阵，其中：
$$A_{ij} = \begin{cases}
1, & v_i v_j \in E \\
0, & \text{否则}
\end{cases}$$

对于加权图，$A_{ij} = w(v_i v_j)$。

**性质**：
- 无向图的邻接矩阵是对称的
- $A^k_{ij}$ 等于从 $v_i$ 到 $v_j$ 长度为 $k$ 的**路径数**

**经典例子**：

```
图：   1 ── 2
      │
      3

邻接矩阵 A = 1 [0 1 1]
             2 [1 0 0]
             3 [1 0 0]

A² = [2 0 0]
     [0 1 1]
     [0 1 1]

A²₁₁ = 2：从 1 到 1 有两条长度为 2 的路径（1-2-1, 1-3-1）
```

#### 3.2.2 关联矩阵

**定义**：$M(G)$ 是一个 $n \times m$ 矩阵，对于无向图：
$$M_{ij} = \begin{cases}
1, & v_i \text{ 与 } e_j \text{ 关联} \\
0, & \text{否则}
\end{cases}$$

**性质**：每列恰好有两个 1（每条边关联两个顶点）。

#### 3.2.3 拉普拉斯矩阵

**定义**：$L(G) = D - A$，其中 $D$ 是度数对角矩阵，$A$ 是邻接矩阵。

$$L_{ij} = \begin{cases}
d(v_i), & i = j \\
-1, & v_i v_j \in E \\
0, & \text{否则}
\end{cases}$$

**核心性质**（谱图理论）：
1. $L$ 是半正定矩阵，特征值均为非负：$0 = \lambda_1 \leq \lambda_2 \leq ... \leq \lambda_n$
2. $\lambda_1 = 0$ 对应的特征向量为全 1 向量
3. $\lambda_2$（代数连通度）> 0 当且仅当图连通
4. $\lambda_2$ 越大，图连通性越强

**Kirchhoff 矩阵树定理**：$L$ 的任意 $n-1$ 阶主子式的行列式等于 $G$ 的生成树数目。

### 3.3 图论经典问题

| 问题 | 描述 | 算法 | 复杂度 |
|------|------|------|--------|
| **最短路径** | 两顶点间权重最小的路径 | Dijkstra / Bellman-Ford / Floyd-Warshall | $O(m \log n) / O(nm) / O(n^3)$ |
| **最大流** | 从源点到汇点的最大流量 | Ford-Fulkerson / Dinic | $O(m \cdot f) / O(\sqrt{m} \cdot m)$ |
| **最小割** | 将图分为两部分的最小边权之和 | 最大流最小割定理 | 等价于最大流 |
| **最大匹配** | 边集中没有公共顶点的最大边子集 | Edmonds / Hopcroft-Karp | $O(n^2 m) / O(\sqrt{n}m)$ |
| **最大团** | 最大完全子图的大小 | Bron–Kerbosch（回溯） | $O(3^{n/3})$（NP-hard） |
| **图着色** | 相邻顶点不同色的最小颜色数 | 贪心 / BFS | NP-hard（一般图） |

## 四、图的几何特性

### 4.1 平面图

**定义**：**平面图**（Planar Graph）是可以画在平面上使得边仅在端点处相交的图。这种画法称为图的**平面嵌入**。

#### 4.1.1 欧拉公式

平面图的最核心定理：

$$V - E + F = 2$$

其中 $V$ 为顶点数，$E$ 为边数，$F$ 为面数（包括外部无限面）。

**推论**：
- 若 $G$ 是 $n \geq 3$ 的连通平面简单图，则 $|E| \leq 3n - 6$
- 若 $G$ 中无三角形，则 $|E| \leq 2n - 4$

**经典例子：柏拉图多面体与欧拉公式**

| 多面体 | 顶点 V | 边 E | 面 F | V−E+F |
|--------|--------|------|------|-------|
| 正四面体 | 4 | 6 | 4 | 2 |
| 立方体 | 8 | 12 | 6 | 2 |
| 正八面体 | 6 | 12 | 8 | 2 |
| 正十二面体 | 20 | 30 | 12 | 2 |
| 正二十面体 | 12 | 30 | 20 | 2 |

#### 4.1.2 Kuratowski 定理

**定理**：图 $G$ 是平面图当且仅当 $G$ 不包含 $K_5$（5 顶点完全图）或 $K_{3,3}$（3+3 完全二分图）的细分。

```
K_5（非平面）:
    A ── B
   / \  /|\
  /   \/ | \
 C ── D  |  E
   \     | /
    \    |/
     ────

K_{3,3}（非平面）:
   A     B     C
  / \   / \   / \
 /   \ /   \ /   \
D     E     F
```

这是平面图最深刻的刻画定理之一，揭示了拓扑性质与组合性质的深刻联系。

### 4.2 图的嵌入

**定义**：图嵌入（Graph Embedding）是将图绘制到一个曲面（如平面、球面、环面）上，使得边仅在顶点处相交。

#### 4.2.1 亏格

**定义**：一个图可以嵌入的最小曲面的**亏格**（Genus）$g(G)$。球面的亏格为 0，环面为 1。

**推广的欧拉公式**（亏格 $g$ 的曲面）：
$$V - E + F = 2 - 2g$$

**经典例子**：
- $K_5$ 和 $K_{3,3}$ 的亏格为 1（可嵌入环面）
- $K_7$ 的亏格为 1
- $K_8$ 的亏格为 2

#### 4.2.2 图的交叉数

**定义**：在平面画法中，交叉数 $\text{cr}(G)$ 是最少的边交叉次数。

**经典结果**：
- $\text{cr}(K_5) = 1$
- $\text{cr}(K_6) = 3$
- $\text{cr}(K_n) \leq \frac{1}{4}\binom{n}{2}\binom{\lfloor n/2 \rfloor}{2}\binom{\lceil n/2 \rceil}{2}$

### 4.3 图的度量几何

#### 4.3.1 距离与直径

**定义**：
- **距离** $d(u,v)$：两顶点之间最短路径的长度（边数或权值和）
- **直径** $\text{diam}(G)$：图中最大距离
- **离心率** $e(v)$：顶点 $v$ 到其他顶点的最大距离
- **中心**：离心率最小的顶点集

**经典例子：超立方体图 $Q_n$**

$Q_n$ 是 $n$ 维立方体的顶点和边构成的图：
- 顶点数：$2^n$
- 顶点标记为 $n$ 位二进制串
- 边连接恰有一位不同的两个串
- 直径：$n$（Hamming 距离的最大值）
- 距离：Hamming 距离
- $Q_n$ 是二分图、正则图（度数为 $n$）

**几何意义**：$Q_n$ 是 $n$ 维超立方体在 $\mathbb{R}^n$ 中的一维骨架（1-skeleton），顶点坐标取 $\{0,1\}^n$，边为欧氏距离为 1 的线段。

#### 4.3.2 图的谱几何

**定义**：谱图理论（Spectral Graph Theory）研究图拉普拉斯矩阵和邻接矩阵的特征值与图几何性质的关系。

**核心结果**：

| 谱性质 | 几何意义 |
|--------|---------|
| $\lambda_2(L)$ > 0 | 图是连通的 |
| $\lambda_2(L)$ 大小 | 图的扩张性——越大越"紧致" |
| $\lambda_n(L)$ | 图的二分性 |
| 特征值分布 | 图的正则性、随机性 |

**Cheeger 不等式**：
$$\frac{\lambda_2}{2} \leq h(G) \leq \sqrt{2\lambda_2}$$

其中 $h(G)$ 是图的 Cheeger 常数（等周常数），衡量图的最小"瓶颈"大小。这一不等式将谱性质（$\lambda_2$）与几何性质（$h(G)$）紧密联系起来。

#### 4.3.3 图距离的几何嵌入

**定义**：将图嵌入到度量空间中，保持顶点间的距离结构（近似）。

**常见嵌入**：

| 嵌入 | 目标空间 | 保持性质 |
|------|---------|---------|
| **等距嵌入** | $\ell_2$ 空间 | $d_G(u,v) = \|f(u)-f(v)\|_2$ |
| **Bourgain 嵌入** | $\ell_1$ 空间 | $O(\log n)$ 失真 |
| **谱嵌入** | $\mathbb{R}^k$（拉普拉斯特征向量） | 保局邻域结构 |
| **力导向布局** | $\mathbb{R}^2$ | 视觉上的美观 |

**经典例子：树的 $\ell_1$ 等距嵌入**

任何树都可以等距嵌入到 $\ell_1$ 空间中：给每条边指定一个坐标方向，每个顶点编码为从根出发的坐标向量。两顶点间的 $\ell_1$ 距离等于它们在树中的距离。

## 五、点集关系

### 5.1 图的点集拓扑

#### 5.1.1 邻域与闭包

| 概念 | 定义 |
|------|------|
| **开邻域** | $N(v) = \{u \in V : uv \in E\}$ |
| **闭邻域** | $N[v] = N(v) \cup \{v\}$ |
| **独立集** | 顶点集 $S$，其中任意两点不相邻 |
| **支配集** | 顶点集 $S$，每个不在 $S$ 中的顶点都与 $S$ 中某顶点相邻 |
| **团** | 顶点集 $S$，其中任意两点都相邻（完全子图） |

#### 5.1.2 顶点覆盖与边覆盖

**顶点覆盖**：顶点集 $C$，使得每条边至少有一个端点在 $C$ 中。

$$\forall e = uv \in E: u \in C \lor v \in C$$

**Kőnig 定理**：在二分图中，最大匹配的大小等于最小顶点覆盖的大小。

**经典例子**：

```
     A ── B ── C
          │
          D

最小顶点覆盖：{B}（一条边覆盖所有边）
最大独立集：{A, C, D}（与顶点覆盖互补）
最大匹配：{A-B, C-B} 或 {A-B, B-D} 等
```

### 5.2 偏序集与哈斯图

**定义**：**偏序集** $(P, \leq)$ 是一个集合 $P$ 和其上的自反、反对称、传递的二元关系 $\leq$。

**哈斯图**（Hasse Diagram）是表示有限偏序集的简化图：
- 若 $x \leq y$，则 $x$ 在 $y$ 下方
- 省略传递边（若 $x \leq y \leq z$ 则不画 $x$ 到 $z$ 的边）
- 省略自反边

**经典例子：子集格的哈斯图**

集合 $\{a,b,c\}$ 的幂集：

```
           {a,b,c}
          /   |   \
     {a,b}  {a,c}  {b,c}
       |   X     X   |
      {a}   {b}   {c}
        \    |    /
           ∅
```

### 5.3 超图

**定义**：**超图**（Hypergraph）$H = (V, \mathcal{E})$ 是图论的推广，其中每条**超边**（hyperedge）$e \in \mathcal{E}$ 是 $V$ 的任意非空子集，不限于二元关系。

$$\mathcal{E} \subseteq \mathcal{P}(V) \setminus \{\emptyset\}$$

**性质**：
- 普通图是每条边恰含 2 个顶点的超图
- 超图可用**关联矩阵** $M_{ij} = 1 \iff v_i \in e_j$ 表示
- 超图的对偶 $H^*$ 将顶点与超边互换

**经典例子：3-一致超图**

$V = \{1,2,3,4\}$, $\mathcal{E} = \{\{1,2,3\}, \{2,3,4\}, \{1,4\}\}$

```
可视化（顶点在面上）：
      1 ─────── 2
      │  ╱ ╲   │
      │ ╱   ╲  │
      │╱     ╲ │
      4 ─────── 3

超边 {1,2,3} 和 {2,3,4} 共享顶点 2,3
超边 {1,4} 是普通边（含 2 个顶点）
```

### 5.4 Ramsey 理论

**定义**：Ramsey 数 $R(s,t)$ 是最小的 $n$，使得任何 $n$ 个顶点的完全图用两种颜色（如红/蓝）染边，必然存在一个红色 $K_s$ 或蓝色 $K_t$。

**经典结果**：
- $R(3,3) = 6$
- $R(3,4) = 9$
- $R(3,5) = 14$
- $R(4,4) = 18$
- $R(5,5)$ 未知（已知范围 $43 \leq R(5,5) \leq 48$）

**经典例子：$R(3,3) = 6$ 的证明**

在 6 个顶点的完全图 $K_6$ 中，任选一个顶点 $v$，它连接另外 5 条边，由鸽笼原理，至少 3 条同色（设为红色）。这 3 条边的另一端若两两之间为红色，则得红色 $K_3$；否则其中有一对为蓝色，则与 $v$ 构成蓝色 $K_3$。

5 个顶点的反例：$K_5$ 的边按五边形（红）+ 五角星（蓝）染色，不含同色 $K_3$。

## 六、图论的经典定理与开放问题

### 6.1 图着色定理

**四色定理**：任何平面图的顶点可用不超过 4 种颜色染色，使得相邻顶点颜色不同。

- 1852 年提出猜想
- 1976 年由 Appel 和 Haken 用计算机证明（首次计算机辅助证明）
- 等价于：任何平面图的色数 $\chi(G) \leq 4$

**五色定理**（较简单）：任何平面图都是 5-可着色的。

### 6.2 图兰定理

**定理**（Turán, 1941）：若 $n$ 个顶点的图不含 $(r+1)$-团 $K_{r+1}$，则最大边数由 Turán 图 $T(n,r)$ 达到——将顶点均分到 $r$ 部分，各部分内部无边，部分之间全连接。

$$\text{ex}(n, K_{r+1}) \leq \left(1 - \frac{1}{r}\right)\frac{n^2}{2}$$

**经典例子**：不含三角形的最大边数（$r=2$）

Turán 图 $T(n,2)$ 是完全二分图 $K_{\lfloor n/2 \rfloor, \lceil n/2 \rceil}$，边数 $\lfloor n^2/4 \rfloor$。即 Mantel 定理：含 $n$ 个顶点的三角形-free 图的最大边数为 $\lfloor n^2/4 \rfloor$。

### 6.3 尚未解决的开放问题

| 问题 | 描述 | 难度 |
|------|------|------|
| **P vs NP** | 图同构属于 P 吗？ | 千禧年大奖难题 |
| **Hadwiger 猜想** | 若图不含 $K_{t}$ 子式，则 $\chi(G) \leq t-1$ | 尚未完全证明 |
| **图同构问题** | 是否存在多项式时间算法判断两图同构？ | Babai 给出拟多项式算法 |
| **最大团问题的精确算法** | 是否存在次指数时间算法？ | 一般认为不可能 |
| **$R(5,5)$ 的精确值** | Ramsey 数 $R(5,5)$ 等于多少？ | 已知 $43 \leq R(5,5) \leq 48$ |

## 七、总结

图与树的理论构成了离散数学的基石，其核心脉络可概括为：

```
欧拉七桥问题（起源）
    │
    ├── 树论（连通、无环、极小图）
    │    ├── 生成树与最小生成树
    │    ├── Prüfer 编码（计数）
    │    └── BST、堆、Trie（数据结构应用）
    │
    ├── 图论（路径、回路、连通性）
    │    ├── 矩阵表示（邻接、拉普拉斯）
    │    ├── 经典问题（最短路径、最大流、匹配）
    │    └── 着色问题（四色定理）
    │
    ├── 几何特性
    │    ├── 平面图与欧拉公式
    │    ├── 图嵌入与亏格
    │    └── 谱图理论（特征值→几何）
    │
    └── 点集关系
         ├── 独立集、支配集、覆盖
         ├── 超图（推广到 n 元关系）
         └── Ramsey 理论（无序中的有序）

            树 = E = V - 1
            图 = 关系集合
            几何 = 嵌入与度量
            点集 = 组合结构
```

树与图的理论不仅为计算机科学提供了算法基础（搜索、网络流、图神经网络），还在物理、化学、生物网络、社交网络分析中有着广泛而深刻的应用。
