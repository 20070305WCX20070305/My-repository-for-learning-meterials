# 图论与常见图算法

## 一、图的基本概念

**图 (Graph)** 由顶点集 \(V\) 和边集 \(E\) 组成，记为 \(G = (V, E)\)。

- **顶点 (Vertex)**：图中的节点
- **边 (Edge)**：顶点之间的连线
- **有向图 (Directed Graph)**：边有方向，记作 \(u \to v\)
- **无向图 (Undirected Graph)**：边无方向，记作 \(u - v\)
- **加权图 (Weighted Graph)**：边带有权重（距离/代价）
- **连通图**：任意两顶点间存在路径
- **环 (Cycle)**：起点与终点相同的路径

### 图的存储方式

| 方式 | 空间复杂度 | 适用场景 |
|------|-----------|---------|
| 邻接矩阵 | O(V²) | 稠密图 |
| 邻接表 | O(V+E) | 稀疏图 |

## 二、图的遍历

### 1. 广度优先搜索 (BFS)

**思路**：从起点出发，逐层向外扩展，借助**队列**实现。先访问距离起点最近的顶点，再访问更远的顶点。

**用途**：无权图最短路、连通分量、拓扑排序（Kahn算法）。

```python
from collections import deque

def bfs(graph, start):
    """
    graph: 邻接表表示的图，如 {0: [1,2], 1: [0,3], ...}
    start: 起始顶点
    """
    visited = set()          # 记录已访问顶点
    queue = deque([start])   # 初始化队列
    visited.add(start)

    while queue:
        node = queue.popleft()      # 出队（队首）
        print(node, end=' ')         # 访问当前节点

        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)  # 未访问的邻居入队
```

### 2. 深度优先搜索 (DFS)

**思路**：从起点出发，沿着一条路径走到底，再回溯，借助**栈**（递归或显式栈）实现。

**用途**：连通分量、环检测、拓扑排序、路径查找。

```python
def dfs_recursive(graph, node, visited=None):
    """递归版 DFS"""
    if visited is None:
        visited = set()

    visited.add(node)
    print(node, end=' ')            # 访问当前节点

    for neighbor in graph[node]:
        if neighbor not in visited:
            dfs_recursive(graph, neighbor, visited)

def dfs_iterative(graph, start):
    """迭代版 DFS (显式栈)"""
    visited = set()
    stack = [start]                 # 用列表模拟栈

    while stack:
        node = stack.pop()          # 出栈（栈顶）
        if node not in visited:
            print(node, end=' ')    # 访问当前节点
            visited.add(node)
            # 栈是后进先出，反向压入保持与递归顺序一致
            for neighbor in reversed(graph[node]):
                if neighbor not in visited:
                    stack.append(neighbor)
```

## 三、最短路径算法

### 1. Dijkstra（迪杰斯特拉）

**思路**：贪心策略，每次从未处理顶点中选择距离起点最近的顶点，用它的出边松弛邻居。借助**优先队列（最小堆）**优化。

**限制**：**不能处理负权边**。

```python
import heapq

def dijkstra(graph, start):
    """
    graph: 邻接表 {u: [(v, weight), ...]}
    start: 起点
    返回: 起点到各顶点的最短距离
    """
    INF = float('inf')
    dist = {node: INF for node in graph}
    dist[start] = 0
    pq = [(0, start)]          # (距离, 顶点)，优先队列

    while pq:
        d, u = heapq.heappop(pq)    # 取出当前距离最小的顶点
        if d > dist[u]:             # 如果该值已无效（旧记录），跳过
            continue

        for v, w in graph[u]:
            new_dist = dist[u] + w
            if new_dist < dist[v]:   # 发现更短路径 → 松弛
                dist[v] = new_dist
                heapq.heappush(pq, (new_dist, v))

    return dist
```

### 2. Bellman-Ford（贝尔曼-福特）

**思路**：对所有边执行 V-1 轮松弛，第 V 轮仍能松弛则说明存在**负权环**。

**优点**：可处理负权边，可检测负环。

```python
def bellman_ford(edges, V, start):
    """
    edges: 边列表 [(u, v, weight), ...]
    V: 顶点数
    start: 起点
    返回: (dist, has_negative_cycle)
    """
    INF = float('inf')
    dist = [INF] * V
    dist[start] = 0

    # 第 1 到 V-1 轮：对每条边进行松弛
    for _ in range(V - 1):
        for u, v, w in edges:
            if dist[u] != INF and dist[u] + w < dist[v]:
                dist[v] = dist[u] + w

    # 第 V 轮：检测负权环
    for u, v, w in edges:
        if dist[u] != INF and dist[u] + w < dist[v]:
            return dist, True   # 存在负权环

    return dist, False
```

### 3. Floyd-Warshall（弗洛伊德）

**思路**：动态规划，`dist[i][j]` 表示 i→j 的最短距离，逐步允许经过更多中间顶点。三重循环，复杂度 O(V³)。

**优点**：一次计算所有顶点对的最短路径；代码极其简洁。

```python
def floyd_warshall(graph):
    """
    graph: 邻接矩阵 (V x V), graph[i][j]=INF 表示无边
    返回: 任意两点间的最短距离矩阵
    """
    V = len(graph)
    INF = float('inf')
    dist = [row[:] for row in graph]  # 拷贝原矩阵

    for k in range(V):         # 允许经过顶点 k
        for i in range(V):
            for j in range(V):
                # i → k → j 比当前 i → j 更短则更新
                if dist[i][k] + dist[k][j] < dist[i][j]:
                    dist[i][j] = dist[i][k] + dist[k][j]

    return dist
```

## 四、最小生成树 (MST)

### 1. Kruskal（克鲁斯卡尔）

**思路**：将所有边按权重排序，从小到大逐个加入，若加入后不形成环则保留。用**并查集**判断是否成环。

```python
class UnionFind:
    """并查集（Disjoint Set Union）"""
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n

    def find(self, x):
        """路径压缩查找"""
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]

    def union(self, x, y):
        """按秩合并"""
        px, py = self.find(x), self.find(y)
        if px == py:
            return False
        if self.rank[px] < self.rank[py]:
            px, py = py, px
        self.parent[py] = px
        if self.rank[px] == self.rank[py]:
            self.rank[px] += 1
        return True

def kruskal(V, edges):
    """
    edges: [(weight, u, v), ...]
    返回: (mst_edges, total_weight)
    """
    edges.sort()                    # 按权重升序排序
    uf = UnionFind(V)
    mst = []
    total = 0

    for w, u, v in edges:
        if uf.union(u, v):          # 不形成环则加入
            mst.append((u, v, w))
            total += w

    return mst, total
```

### 2. Prim（普里姆）

**思路**：从任意顶点出发，不断将**连接已选集合与未选集合的最小权重边**加入，借助优先队列实现。

```python
import heapq

def prim(graph, start=0):
    """
    graph: 邻接表 {u: [(v, weight), ...]}
    start: 起始顶点
    返回: (mst_edges, total_weight)
    """
    n = len(graph)
    visited = [False] * n
    pq = [(0, start, -1)]       # (权重, 当前顶点, 父顶点)
    mst = []
    total = 0

    while pq:
        w, u, parent = heapq.heappop(pq)
        if visited[u]:
            continue
        visited[u] = True
        total += w
        if parent != -1:
            mst.append((parent, u, w))

        for v, weight in graph[u]:
            if not visited[v]:
                heapq.heappush(pq, (weight, v, u))

    return mst, total
```

## 五、拓扑排序（Topological Sort）

**适用条件**：**有向无环图 (DAG)**。

### Kahn 算法（BFS 思路）

**思路**：统计每个顶点的入度，将入度为 0 的顶点入队；逐个出队并减少邻居入度，入度变为 0 则入队。最终若输出顶点数 < V，说明存在环。

```python
from collections import deque

def topological_sort(graph, V):
    """
    graph: 邻接表 {u: [v, ...]}  (有向边 u → v)
    V: 顶点数
    返回: 拓扑序列（若存在环则返回空列表）
    """
    in_degree = [0] * V
    for u in graph:
        for v in graph[u]:
            in_degree[v] += 1

    q = deque([i for i in range(V) if in_degree[i] == 0])
    result = []

    while q:
        u = q.popleft()
        result.append(u)
        for v in graph[u]:
            in_degree[v] -= 1
            if in_degree[v] == 0:
                q.append(v)

    return result if len(result) == V else []
```

## 六、环检测 (Cycle Detection)

### 有向图 — DFS + 三色标记

**思路**：DFS 遍历时，给顶点标记三种状态：
- **0 (未访问)**：尚未搜索
- **1 (正在访问)**：在当前 DFS 路径上（在栈中）
- **2 (已访问完)**：该顶点及其子树搜索完毕

若 DFS 过程中遇到状态为 1 的邻居，说明存在环。

```python
def has_cycle_directed(graph, V):
    """
    检测有向图是否存在环
    """
    state = [0] * V   # 0=未访问, 1=访问中, 2=已结束

    def dfs(u):
        if state[u] == 1:       # 碰到当前路径上的顶点 → 有环
            return True
        if state[u] == 2:       # 已处理完毕，跳过
            return False

        state[u] = 1            # 标记为"访问中"
        for v in graph[u]:
            if dfs(v):
                return True
        state[u] = 2            # 标记为"已结束"
        return False

    for i in range(V):
        if state[i] == 0:
            if dfs(i):
                return True
    return False
```

### 无向图 — DFS + 父节点

**思路**：DFS 遍历，如果遇到已访问的邻居**且不是当前节点的父节点**，说明存在环。

```python
def has_cycle_undirected(graph, V):
    """检测无向图是否存在环"""
    visited = [False] * V

    def dfs(u, parent):
        visited[u] = True
        for v in graph[u]:
            if not visited[v]:
                if dfs(v, u):
                    return True
            elif v != parent:   # 访问到非父节点的已访问顶点 → 有环
                return True
        return False

    for i in range(V):
        if not visited[i]:
            if dfs(i, -1):
                return True
    return False
```

## 七、并查集 (Disjoint Set Union / Union-Find)

**思路**：高效管理元素所属集合，支持**查询**（Find）和**合并**（Union）。优化手段：路径压缩 + 按秩合并，均摊 O(α(n))，α 为阿克曼反函数（近似常数）。

```python
class DSU:
    def __init__(self, n):
        self.parent = list(range(n))
        self.size = [1] * n     # 集合大小（也可用 rank）

    def find(self, x):
        """路径压缩：将 x 直接指向根节点"""
        while self.parent[x] != x:
            self.parent[x] = self.parent[self.parent[x]]  # 隔代压缩
            x = self.parent[x]
        return x
        # 递归写法：if self.parent[x] != x: self.parent[x] = self.find(self.parent[x])

    def union(self, a, b):
        """按大小合并：小树挂到大树下"""
        pa, pb = self.find(a), self.find(b)
        if pa == pb:
            return False
        if self.size[pa] < self.size[pb]:
            pa, pb = pb, pa
        self.parent[pb] = pa
        self.size[pa] += self.size[pb]
        return True

    def connected(self, a, b):
        """判断两顶点是否连通"""
        return self.find(a) == self.find(b)
```

## 八、算法复杂度速查表

| 算法 | 时间复杂度 | 空间复杂度 | 核心数据结构 |
|------|-----------|-----------|-------------|
| BFS | O(V+E) | O(V) | 队列 |
| DFS | O(V+E) | O(V) | 栈 / 递归 |
| Dijkstra | O((V+E)logV) | O(V+E) | 优先队列 |
| Bellman-Ford | O(VE) | O(V) | 数组 |
| Floyd-Warshall | O(V³) | O(V²) | 二维数组 |
| Kruskal | O(ElogE) | O(V) | 并查集 |
| Prim | O((V+E)logV) | O(V) | 优先队列 |
| 拓扑排序 (Kahn) | O(V+E) | O(V) | 队列 |
| 并查集 (优化) | O(α(n)) 均摊 | O(n) | 数组 |

## 九、应用场景总结

|---------|---------|

| 动态连通性 | 并查集 |
| 二分图判定 | BFS / DFS 染色法 |
