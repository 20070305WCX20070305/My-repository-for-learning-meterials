# 树——根深叶茂

## 术语

非线性数据结构，节点，层级关系

### 节点，树的概念

1. 值，子节点，父节点，度（子节点数），深度，高度（到叶节点最大距离）
2. 根节点，叶节点，层级，子树，节点关系（祖先，父节点，子节点，兄弟节点，后代节点）

### 分类

1. 普通树
2. 二叉树：满二叉树，平衡二叉树（左右子树高度差小于等于1），二叉搜索树，完全二叉树（只有最后一行不满，且节点从左向右排列）
3. 特殊树：平衡搜索树，前缀树，堆，B树

### 操作

1. 插入
2. 删除
3. 查找
4. 遍历：
    - 深度优先（DFS）
      - 前序
      - 中序
      - 后序

    - 广度优先（BFS）

from collections import deque

def level_order(root):
    if root is None:
        return

    q = deque([root])
    while q:
        node = q.popleft()
        print(node.data, end="")
        child = node.firstchild

        while child:
            q.append(child)

### BST：

二叉搜索树，二叉检索树，二叉查找树，二叉排序树，有序二叉树，BST，是一个东西

#### 中序遍历，整体思路与前面类似，只是写法更加清晰，DFS+递归

```python
from typing import Optional, List # 说明变量类型

# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    # 求解中序表达式
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        result = []
        
        def dfs(node: Optional[TreeNode]):
            if not node:
                return
            dfs(node.left)
            result.append(node.val)
            dfs(node.right)
        
        dfs(root)
        return result

    
```

#### 题目

第一行是一个整数n，表示二叉树的结点个数。二叉树结点编号从1到n，根结点为1，n <= 10 接下来有n行，依次对应二叉树的n个节点。 每行有两个整数，分别表示该节点的左儿子和右儿子的节点编号。如果第一个（第二个）数为-1则表示没有左（右）儿子

```python
#只写构建树的过程

class TreeNode:
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None

def build_tree(relation: list)::
    tree_nodes = [None] * (len(relation) + 1)
    for i in range(1, len(relation) + 1):
        tree_nodes[i] = TreeNode[i]

    for i in range(0, len(relation) + 1):
        if relation[i][0] != -1:
            tree_nodes[i].left = tree_nodes[relation[i][0]]

        if relation[i][1] != -1:
            tree_nodes[i].right = tree_nodes[relation[i][1]]
    return tree_nodes

```

#### 题目：判断两棵树是否相同

```python

def is_same_tree(p, q):
    if not p and not q:
        return True
    if not p or not q:
        return False

    return (p.data == q.data) and (is_same_tree(p.left, q.left)) and (is_same_tree(p.right, q.right))

```

    return maxvalue(root.right)
