## 解析树与抽象语法树 AST

1. 词法分析（Lexical Analysis）：此步骤中，输入的字符序列（代码、表达式等）被转换成一系列标记（tokens），例如数字、操作符、括号等。

2. 语法分析（Syntactic Analysis）：在该阶段，标记序列根据语法规则被组织成解析树。语法分析器会使用诸如递归下降、LL 解析、LR 解析等算法来构造树。

3. 语义分析（Semantic Analysis）：构建的解析树可能会进一步经过语义检查，以确保表达式的意义是合理的，比如类型检查等。


### 完全括号表达式

4种标记：左括号、右括号、运算符和操作数。

左括号代表新表达式的起点，所以应该创建一棵对应该表达式的新树。反之，遇到右括号则意味着到达该表达式的终点。操作数既是叶子节点，也是其运算符的子节点。此外，每个运算符都有左右子节点。

四条规则（对于一个运算过程模拟一下就行了，比较好理解）：

1. 如果当前标记是(，就为当前节点添加一个左子节点，并下沉至该子节点；

2. 如果当前标记在列表['+', '-', '/', '＊']中，就将当前节点的值设为当前标记对应的运算符；为当前节点添加一个右子节点，并下沉至该子节点；

3. 如果当前标记是数字，就将当前节点的值设为这个数并返回至父节点；

4.  如果当前标记是)，就跳到当前节点的父节点。

class Stack:  # 定义栈（可以通用）
    def __init__(self):
        self.items = []
        self.stacksize = 0

    def is_empty(self):
        return self.stacksize == 0

    def push(self, newitem):
        self.items.append(newitem)
        self.stacksize += 1

    def pop(self):
        self.stacksize -= 1
        return self.items.pop()

    def peek(self):
        return self.items[self.stacksize - 1]

    def size(self):
        return self.stacksize

### 调度场算法

两个结构：
- 输出队列
- 运算符栈：临时存放运算符，函数名与左括号

|Token类型   | 动作   |
|-----------|------------|
|数字/操作数|直接加入输出队列。|
|函数名  | 压入运算符栈（视为特殊的“一元前缀运算符”）。|
|逗号    | 依次弹出运算符栈顶的运算符并输出，直到遇到左括号。用于分隔函数参数。|
|运算符（+ - * / ^ 等）|当栈顶存在优先级更高或同级的运算符（结合性决定行为）时，弹出栈顶并输出；之后将当前运算符压栈。|
|左括号 ( |压入运算符栈。|
|右括号 ) |不断弹出栈顶运算符并输出，直到遇到左括号；弹出左括号但不输出。如果栈顶是函数名，也要弹出并输出（表示函数结束）。|

运算符优先级与结合性
|运算符|优先级|结合性|
|---|---|---|
|二元`or`|1|左结合|
|二元`and`|2|左结合|
|加减号`+``-`|3|左结合|
|乘除号`*``/`|4|左结合|
|一元`not`, `A`……|5|右结合|
|乘方`**`|6|右结合|

- 左结合运算符（如 +）弹出时要求栈顶优先级 **大于或等于** 当前运算符；
- 右结合运算符（如 ^）弹出时要求栈顶优先级 **严格大于** 当前运算符。

#### 调度场算法的python实现

- 本质对象：有层级结构的二元算符（比如+-*/^，优先级不同），一元算符（用单个大写字母表示A、B、C，作用于后面的数字）
- 假设表达式可以转化为数值与运算符构成的列表，也就是可以使用`input().split()`

1. 二元算符：+、-、*、/、and、or
2. 一元算符：not、A、B……
3. 括号，操作数（整数或者浮点数）

**算法实现**

**最最关键的部分！！！**

- 核心结构：输出队列`output`（存放token，用于最终输出），运算符栈`op_stack`（暂存运算符与左括号）
- token的处理：
  1. 数字：直接放入`output`
  2. 左括号：压入`op_stack`
  3. 右括号：弹出`op_stack`中的运算符，直到出现左括号
  4. 运算符：首先得到运算符的优先级与结合性，栈顶满足弹出条件时弹出，将当前运算符压入`op_stack`
  5. 输入结束：将栈内部剩余元素弹出，逐个加在`output`后面。

- 后缀表达式的求值计算：
  1. 将数字加入数值栈`num_stack`
  2. 遇到一个一元算符，弹出一个`a`，计算`operator(a)`，再压入栈
  3. 遇到二元算符，弹出两个`a`和`b`，计算`b operator a`，数值再压入栈
