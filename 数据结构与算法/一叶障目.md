### 带度数的链表

两棵树的层次遍历序列分别为：C E F G K H J / D X I

每个结点对应的度数为：3 3 0 0 0 0 0 / 2 0 0

我们将以上序列存储起来，就可以在以后的应用中恢复这个森林。在存储中，我们可以将第一棵树表示为C 3 E 3 F 0 G 0 K 0 H 0 J 0，第二棵树表示为D 2 X 0 I 0。

现在有一些通过带度数的层次遍历序列存储的森林数据，为了能够对这些数据进行进一步处理，首先需要恢复他们。

```python
class Node:
    def __init__(self, value, degree):
        self.value = value
        self.degree = degree
        self.children = []

# 根据带度数的层次序列构造树
def build_tree(tokens):
    # tokens 的格式：[字母, 度数, 字母, 度数, ...]
    # 第一个结点为根
    root = Node(tokens[0], int(tokens[1]))
    queue = deque([root])
    index = 2  # 下一个待处理的token索引
    while queue and index < len(tokens):
        current = queue.popleft()
        # current.degree 个孩子依次出现在 tokens 中
        for _ in range(current.degree):
            # 每个孩子由两个元素构成：字母和度数
            child = Node(tokens[index], int(tokens[index+1]))
            current.children.append(child)
            queue.append(child)
            index += 2
    return root
```

### 二叉树的数组表示

- 完美二叉树：节点索引为$i$，那么左子节点索引$2i+1$，右子节点索引为$2i+2$。
- 任意二叉树：显式的写出所有的`None`即可

数组表述中，数组天然的就是层序遍历

```python
class ArrayBinaryTree:
    def __init__(self, array: list):
        self.tree = array

    def size(self):
        return len(self.tree)
    
    def value(self, i):
        if i < 0 or i >= self.size():
            return None
        return self.tree[i]

    def left(self, i):
        return 2 * i + 1
    
    def right(self, i):
        return 2 * i + 2
    
    def parnent(self, i):
        return (i - 1) // 2

    def level(self):
        token = list()
        for i in range(self.size()):
            if self.tree[i] is not None:
                token.append(self.tree[i])
        return token

    def dfs(self, index: int, order: str):
        if self.value(index) is None:
            return 
        
        if order == "pre":
            self.res.append(self.value(index))

        self.dfs(self.left(index), order)

        if order == "mid":
            self.res.append(self.value(index))

        self.dfs(self.right(index), order)

        if order == "post":
            self.res.append(self.value(index))

    def preorder(self):
        self.res = []
        self.dfs(0, order = "pre")
        return self.res

    def midorder(self):
        self.res = []
        self.dfs(0, order = "mid")
        return self.res
            
    def postorder(self):
        self.res = []
        self.dfs(0, order = "post")
        return self.res

```
