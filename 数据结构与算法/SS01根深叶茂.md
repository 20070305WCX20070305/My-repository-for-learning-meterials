# 树——根深叶茂

## 术语

非线性数据结构，节点，层级关系

### 节点，树的概念

1. 值，子节点，父节点，度（子节点数），深度，高度（到叶节点最大距离）
2. 根节点，叶节点，层级，子树，节点关系（祖先，父节点，子节点，兄弟节点，后代节点）

### 分类

1. 普通树
2. 二叉树：满二叉树，平衡二叉树（左右子树高度差小于等于1），二叉搜索树，完全二叉树（只有最后一行不满，且节点从左向右排列）
3. 特殊树：平衡搜索树，前缀树，堆，B树

### 操作

1. 插入
2. 删除
3. 查找
4. 遍历：
    - 深度优先（DFS）
      - 前序
      - 中序
      - 后序

    - 广度优先（BFS）

#### 中序遍历，整体思路与前面类似，只是写法更加清晰，DFS+递归

```python
from typing import Optional, List # 说明变量类型

# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    # 求解中序表达式
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        result = []
        
        def dfs(node: Optional[TreeNode]):
            if not node:
                return
            dfs(node.left)
            result.append(node.val)
            dfs(node.right)
        
        dfs(root)
        return result

    
```

#### 判断平衡二叉树

思路：首先构建树类，然后写一个程序计算树的高度，最后递归判断是否是平衡二叉树即可。
但是这样不是最优解

最优解：**重新定义高度函数，在非平衡时直接定义成-1，遇到-1直接将-1向上传递，平衡时是正常的高度**

```python

class TreeNode:
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None
    
def is_balanced_binarytree(root):
    def hight(root):
        if not root:
            return 0
        left_h = hight(root.left)
        right_h = hight(root.right)

        if (left_h == -1) or (right_h == -1) or abs(right_h - left_h) > 1:
            return -1
        return max(left_h, right_h) + 1
    
    return hight(root) != -1

```

## m-叉树的性质

1. **<mark>总结点数</mark>**：设树中总的结点数为 $N$，则：
   $$
   N = N_0 + N_1 + N_2 + \dots + N_m
   $$
   其中 $N_0$ 表示度数为 0 的结点数（即叶结点），$N_1, N_2, \dots, N_m$ 分别表示度数为 1, 2, ..., $m$ 的结点数。

2. **<mark>总分支数</mark>**：在树中，每个度数为 $k$ 的结点贡献了 $ k $ 条分支。因此，树中的总分支数可以表示为：
   $$
   \text{总分支数} = 1 \cdot N_1 + 2 \cdot N_2 + \dots + m \cdot N_m
   $$

3. **树的总分支数与总结点数的关系**：对于一棵树，总分支数等于总结点数减去 1（因为除了根结点外，每个结点都由一条分支连接到其父结点）。因此：
   $$
   \text{总分支数} = N - 1
   $$

> 将以上关系结合起来，可以得到关于 $N_0$ 的方程。



<mark>对任何一棵非空二叉树，如果叶节点数 $n_0$，度为2的非叶节点数 $n_2$，则：</mark>

<mark>$n_0 = n_2 + 1$</mark>

> 在一棵二叉树中，除了度为0的叶子节点，就是度为1的节点和为2的节点，则树的总节点数为：
>
> $N = n_0 + n_1 + n_2$
>
> 在二叉树中节点总数为 $N$ 和边数之间的关系是：<mark>边数 = $N - 1$</mark>，所以有：
>
> $n_0 + n_1 + n_2 - 1 = 2n_2 + n_1$ 
>
> 最后得到：$n_0 = n_2 + 1$
