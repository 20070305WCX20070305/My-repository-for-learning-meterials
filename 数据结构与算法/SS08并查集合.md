# 并查集合

通常情况下，使用<mark>邻接表<mark>来表示<mark>并查集（Disjoint Set）和前缀树（Trie Tree）<mark>是比较常见的做法。

并查集（Disjoint Set）：在并查集中，每个元素都属于一个集合，并且这些集合之间是不相交的。为了高效地实现并查集操作，通常会使用树形结构来表示集合之间的关系。<mark>每个集合可以用一个树表示，其中树的根节点是集合的代表元素。<mark>使用邻接表来表示这种树形结构是一种常见的做法，其中每个节点存储其父节点的指针。

前缀树（Trie Tree）：前缀树是一种用于存储字符串集合的数据结构，通常用于快速地进行字符串匹配和搜索。在前缀树中，每个节点代表一个字符，从根节点到叶子节点的路径表示一个字符串。为了表示字符串的结构，<mark>通常会使用邻接表来表示前缀树，其中每个节点存储一个字符以及指向子节点的指针列表。<mark>

使用邻接表表示并查集和前缀树可以高效地支持各种操作，如并查集的合并和查找操作，以及前缀树的插入、搜索和删除操作。同时，邻接表也能够比较直观地表示数据结构之间的关系，便于理解和实现。

## 并查集

一种存储非重叠或不相交子集的数据结构被称为不相交集合数据结构（Disjoint Set Data Structure）。不相交集合数据结构支持以下操作：

- 向不相交集合中添加新集合。
- 使用**Union（合并）**操作将不相交集合合并为一个集合。
- 使用**Find（查找）**操作找到不相交集合的代表元素。
- 检查两个集合是否为不相交。

考虑一个包含若干人的情景，并需要对他们执行以下任务（并查集的应用）：

- 添加新的朋友关系，即一个人 x 成为另一个人 y 的朋友（即将新元素添加到集合中）。
- 查找个体 x 是否是个体 y 的朋友（直接或间接的朋友）。

### 使用到的数据结构

1. 数组： 一个名为Parent[]的整数数组。如果我们处理的是N项，数组的第i个元素代表第i项。更具体地说，Parent[]数组的第i个元素是第i项的父节点。这些关系创建了一个或多个虚拟树。

2. 树： 它是一个不相交集。如果两个元素位于同一棵树中，则它们位于同一个不相交集中。每棵树的根节点（或最顶部的节点）被称为该集合的代表。每个集合总是有一个唯一的代表。识别代表的一个简单规则是，如果‘i’是某个集合的代表，则Parent[i] = i。如果‘i’不是他所在集合的代表，则可以通过沿树向上遍历直到找到代表为止。


### 并查集的实现
- `find(x)`: 查找根节点（路径压缩优化）
- `union(x, y)`: 合并两个集合
- 应用场景：分组问题、连通性判断（如宗教信仰统计）

```python

class DSU:
    def __init__(self, n: int):
        self.parent = list(range(n))
        self.size = [1] * n    #size[i] 表示以i为根的集合的大小

    def find(self, x):
        while self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])  # 隔代压缩
            x = self.parent[x]
        return x

    def union(self, x, y):
        rx, ry = self.find(x), self.find(y)
        if rx == ry:
            return 
        if self.size(rx) < self.size(ry):  # 大树挂小树
            rx, ry = ry, rx
        self.find[ry] = rx
        self.size[rx] = self.size[ry] + self.size[rx]

    def connected(self, x, y):
        return self.find(x) == self.find(y)

```

## 加权并查集


普通并查集只能回答“两个元素在不在同一个集合里”。带权并查集可以回答“两个元素在同一集合内，<mark>它们之间的相对关系是什么？”<mark>
比如：

- 食物链：A 吃 B，B 吃 C，问 A 和 C 的关系。
- 区间和：已知若干个区间 [l,r] 的和，判断某个新区间和是否冲突。
- 距离/排名：已知 x 比 y 高 3 分，y 比 z 高 2 分，问 x 和 z 的分数差。

### 核心思想：每个节点储存它到父节点的权值

设 parent[x] 是 x 的父节点，weight[x] 表示 x 到 parent[x] 的相对关系（这个“关系”根据具体问题定义，比如差值、距离、模3关系等）。

1. Find 操作（路径压缩）：当我们要找 x 的根时，我们会把 x 直接挂到根上。但此时 x 到根的关系会变，必须更新 weight[x]。原本：x -> p -> root，已知 weight[x] 是 x→p 的关系，weight[p] 是 p→root 的关系。路径压缩后：x -> root，新的 weight[x] 应该是 x→p 合并 p→root。合并方式取决于关系的定义，最常见的是加法（如差值）或模加法（如食物链的模 3）。

公式：<mark>weight[x] = weight[x] + weight[p]<mark>（然后 parent[x] = root）。

2. Union 操作（合并两个集合）：假设我们要合并 x 和 y，且已知它们之间的相对关系 rel(x, y)（比如 x 比 y 大 d）。我们先找到 rootX = Find(x)，rootY = Find(y)。如果 rootX != rootY，就把 rootX 挂到 rootY 下（或反过来），并设置 weight[rootX]。

3. 怎么计算 weight[rootX]？已知 x→rootX（即 weight[x]），y→rootY（即 weight[y]），以及要求的 x→y 关系 d。
我们希望：x → rootX → rootY 这条路径上的总关系，应该等于 x → y → rootY 的总关系。所以：<mark>weight[x] + weight[rootX] = d + weight[y]<mark>。解得：weight[rootX] = d + weight[y] - weight[x]

这种“向量式”的理解是带权并查集最通用的方法：把关系看作有向边上的值，合并时用加减法解出未知边权。

### python实现

```python

class WeightedDSU:
    def __init__(self, n):
        self.parent = list(range(n))
        self.weight = [0] * n
        self.size = [1] * n

    def find(self, x):
        """查找根节点，同时路径更新，weight[x]变为x到根节点的权重。"""
        if self.parent[x] != x:
            orign_parent = self.parent[x]
            self.parent[x] = self.find(self.parent[x])
            self.weight[x] += self.weight[orign_parent]
        return self.parent[x]

    def union(self, x, y, delta):
        """weight[y]-weight[x]=delta，x到y的权重差为delta，合并x，y同时更新权重。"""
        rx, ry = self.find(x), self.find(y)
        if rx == ry:
            return 
        if self.size[rx] < self.size[ry]: # 交换保证树不退化，但是因为有向，x，y都要变
            rx, ry = ry, rx
            x, y = y, x
            delta = - delta

        self.parent[ry] = rx
        self.size[rx] += self.size[ry]
        self.weight[ry] = self.weight[x] + delta - self.weight[y]

    def diff(self, x, y):
        rx, ry = self.find(x), self.find(y)
        if rx != ry:
            raise ValueError("Not Connected!")
        return self.weight[x] - self.weight[y]

```


### 食物链

动物王国中有三类动物A,B,C，这三类动物的食物链构成了有趣的环形。A吃B， B吃C，C吃A。 现有N个动物，以1－N编号。每个动物都是A,B,C中的一种，但是我们并不知道它到底是哪一种。 有人用两种说法对这N个动物所构成的食物链关系进行描述： 第一种说法是"1 X Y"，表示X和Y是同类。 第二种说法是"2 X Y"，表示X吃Y。 此人对N个动物，用上述两种说法，一句接一句地说出K句话，这K句话有的是真的，有的是假的。当一句话满足下列三条之一时，这句话就是假话，否则就是真话。 1） 当前的话与前面的某些真的话冲突，就是假话； 2） 当前的话中X或Y比N大，就是假话； 3） 当前的话表示X吃X，就是假话。 你的任务是根据给定的N（1 <= N <= 50,000）和K句话（0 <= K <= 100,000），输出假话的总数。

输入：第一行是两个整数N和K，以一个空格分隔。 以下K行每行是三个正整数 D，X，Y，两数之间用一个空格隔开，其中D表示说法的种类。 若D=1，则表示X和Y是同类。 若D=2，则表示X吃Y。

输出：只有一个整数，表示假话的数目。

```python
class WeightedDUS:
    def __init__(self, n):
        self.parent = list(range(n))
        self.weight = [0] * n
        self.size =  [1] * n
        
    def find(self, x):
        if self.parent[x] != x:
            origin_parent = self.parent[x]
            self.parent[x] = self.find(origin_parent)        
            self.weight[x] = (self.weight[x] + self.weight[origin_parent]) % 3
        return self.parent[x]
    
    def union(self, x, y, delta):
        rx, ry = self.find(x), self.find(y) 
        if rx == ry:
            return 
        if self.size[rx] < self.size[ry]:
            rx, ry = ry, rx
            x, y = y, x
            delta = (- delta) % 3
            
        self.parent[ry] = rx
        self.size[rx] += self.size[ry]
        self.weight[ry] = (self.weight[x] - delta - self.weight[y]) % 3
        
    def diff(self, x, y):
        rx, ry = self.find(x), self.find(y)
        return (self.weight[x] - self.weight[y]) % 3
    
def main():
    n, m = map(int, input().split())
    data = list()
    for _ in range(0, m):
        data.append(list(map(int, input().split())))
    
    dus = WeightedDUS(n) 
    false = 0
    for items in data:
        a, b, c = items
        if a == 2 and b == c:
            false += 1
            continue
        elif b > n or c > n:
            false += 1
            continue
        if a == 1:
            rx, ry = dus.find(b), dus.find(c)
            if rx == ry and dus.diff(b, c) != 0:
                false += 1
                continue
            dus.union(b, c, 0)
            
        if a == 2:
            rx, ry = dus.find(b), dus.find(c)
            if rx == ry and dus.diff(b, c) != 1:
                false += 1
                continue
            dus.union(b, c, 1)
            
    print(false)
    
if __name__ == "__main__":
    main()
```
